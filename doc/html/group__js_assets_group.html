NATS_EXTERN natsStatus natsJS_AddStream ( natsJSStreamInfo **  si,
natsJS *  js,
natsJSStreamConfig *  cfg,
natsJSOptions *  opts,
natsJSErrCode *  errCode 
)

Creates a stream based on the provided configuration (that cannot be NULL). The name is mandatory and cannot contain . characters.

Note
If you do not need a natsJSStreamInfo to be returned, you can pass NULL, otherwise, on success you are responsible for freeing this object.

cfgthe pointer to the natsJSStreamConfig.
optsthe pointer to the natsJSOptions object, possibly NULL.
errCodethe location where to store the JetStream specific error code, or NULL if not needed.
